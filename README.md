Timetable Field (Drupal 8 module)
=================================

Drupal 8 module - Simple timetable field that can be display through an nice weekly grid.

Requirements:
- Drupal 8 (>= 8.1.x)

[Try out a demonstration](http://www.budaorsiboxclub.hu/orarend)
